# what to use when confused for measure vs column .
Check Error = IFERROR(Fact_Sales[TotalSales]/0,"error")
here all rows will show error as message in check error column in fact sales table  as /0 will give error 

If we use a Column: Power BI calculates the error once and stores it. It takes up a tiny bit of memory/file size.

If we use a Measure: Power BI calculates the error every time you look at the screen. Since it's a simple /0 check, it's fast, but it's technically "extra work" for the computer to do repeatedly for a result that never changes.
